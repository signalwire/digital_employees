# Example (Starter Template)

A starter template for building AI agents with WebRTC capabilities using SignalWire's platform. Use this as a foundation for creating your own conversational AI agents.

## Live Demo

Try it online: https://example.signalwire.io

## Source Repository

The full source code is maintained in a separate repository:

**[github.com/signalwire-demos/example](https://github.com/signalwire-demos/example)**

## Features

- **Voice AI Interactions** - Real-time conversational capabilities
- **WebRTC Communication** - Audio/video streaming
- **SWAIG Integration** - SignalWire AI Gateway support
- **Clean Architecture** - Well-structured starting point for custom agents

## Architecture

- **Backend**: Python with FastAPI
- **Frontend**: JavaScript, CSS, HTML
- **Communication**: WebRTC for real-time media
- **Platform**: SignalWire AI infrastructure
