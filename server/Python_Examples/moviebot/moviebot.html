Welcome to the MovieBot SWAIG Server

This server provides various endpoints to interact with The Movie Database (TMDb) API.

Available Functions:

  • Search for movies by title
    Function search_movie
    Parameters query (required), language, page, include_adult, region, year, primary_release_year
    Description Searches for movies using a title query and returns results based on specified criteria.
  • Retrieve detailed information about a movie
    Function get_movie_details
    Parameters movie_id (required), language, append_to_response
    Description Retrieves comprehensive details about a specific movie using its TMDb ID.
  • Discover movies by different criteria
    Function discover_movies
    Parameters language, region, sort_by, include_adult, include_video, page, primary_release_year, primary_release_date_gte, primary_release_date_lte, with_genres, with_cast, with_crew, with_keywords, with_runtime_gte, with_runtime_lte
    Description Allows discovery of movies based on various filters and sorting options.
  • Get trending movies
    Function get_trending_movies
    Parameters time_window, language, page
    Description Retrieves a list of movies that are currently trending within a specified time window.
  • Get movie recommendations
    Function get_movie_recommendations
    Parameters movie_id (required), language, page
    Description Provides movie recommendations based on a specific movie's TMDb ID.
  • Retrieve the list of official genres
    Function get_genre_list
    Parameters language
    Description Retrieves a list of official movie genres available in TMDb.
  • Get upcoming movies
    Function get_upcoming_movies
    Parameters language, region, page
    Description Retrieves a list of movies that are soon to be released.
  • Get movies currently playing in theaters
    Function get_now_playing_movies
    Parameters language, region, page
    Description Retrieves a list of movies that are currently playing in theaters.
  • Retrieve similar movies
    Function get_similar_movies
    Parameters movie_id (required), language, page
    Description Retrieves a list of movies similar to a specified movie.
  • Search for movies, TV shows, and people in a single request
    Function multi_search
    Parameters query (required), language, page, include_adult, region
    Description Performs a search across movies, TV shows, and people using a single query.
  • Retrieve cast and crew information for a movie
    Function get_movie_credits
    Parameters movie_id (required), language
    Description Retrieves detailed cast and crew information for a specific movie.
  • Retrieve detailed information about a person
    Function get_person_details
    Parameters person_id (required), language, append_to_response
    Description Retrieves comprehensive details about a person using their TMDb ID.

For more information, visit the GitHub repository.
